# React Todo App

Aplikasi Todo List sederhana menggunakan React Functional Component dan Hooks.

## Fitur
- Menampilkan daftar todo
- Menambahkan todo baru
- Menandai todo sebagai selesai
- Menghapus todo

## Teknologi
- React
- JavaScript 
- CSS

## Alur Aplikasi
1. User mengetik todo pada input.
2. Klik tombol **Tambah** untuk menyimpan todo ke state.
3. Todo ditampilkan dalam list.
4. Klik teks todo untuk menandai selesai.
5. Klik tombol ‚ùå untuk menghapus todo.
