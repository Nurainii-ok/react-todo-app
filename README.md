# React Todo App

## Deskripsi
React Todo App adalah aplikasi Todo List sederhana yang dibuat menggunakan React. Aplikasi ini memungkinkan pengguna untuk menambahkan todo, menandai todo sebagai selesai, dan menghapus todo.

Project ini dibuat menggunakan React Functional Component dan Hooks (`useState`) untuk mengelola state aplikasi.


## Fitur
- Menampilkan daftar todo
- Menambahkan todo baru
- Menandai todo sebagai selesai
- Menghapus todo
- UI sederhana dan mudah digunakan


## Teknologi yang Digunakan
- React JS
- JavaScript
- React Hooks (`useState`)


## Alur Aplikasi
1. User memasukkan teks todo melalui input.
2. Data todo disimpan ke dalam state menggunakan `useState`.
3. Todo ditampilkan dalam bentuk list.
4. User dapat:
   - Menandai todo sebagai selesai dengan mengklik teks todo.
   - Menghapus todo dengan tombol hapus.
5. Setiap perubahan data akan langsung memperbarui tampilan aplikasi.
